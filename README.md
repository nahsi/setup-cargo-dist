# setup-cargo-dist

Setup [cargo-dist](https://github.com/axodotdev/cargo-dist).

## Usage

```yaml
- name: Setup cargo-dist
  uses: nahsi/setup-cargo-dist@v1
  with:
    # Default: "latest"
    version: 0.0.2
```
